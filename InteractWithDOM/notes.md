# Interact with the Document Object Model 

## The Dom Tree 

### window 

global window object. 

For example: 

```
window.innerWidth

```

### document

e.g: 

```
document.getElementById("")

```
### body 

e.g:

```
<body>

```

e.g: 

```
document.body

```
